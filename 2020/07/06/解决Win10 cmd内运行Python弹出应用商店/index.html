解决Win10 cmd内运行Python弹出应用商店

问题描述

我的Python安装了两个版本，一个是2.7版本，一个是3.5版本。为了区分，在 cmd 内输入 python 运行的是2.7版本，输入 python3 运行的是3.5版本，今天运行的时候发现2.7版本可以在 cmd 中正常运行，而一输入 python3 就自动打开Windows应用商店。

解决办法


（1）选中这个Path，点编辑，有些小伙伴会发现在环境变量里有一行：”C:\Users\Joe\AppData\Local\Microsoft\WindowsApps”，通过右边的上移下移，把Python的环境变量优先级调高：

（2）完成这一步有些小伙伴已经可以在cmd中正常启动Python了，如果还不行，接着往下看：

（3）单击红框内的Path（注意这个Path和第一步的Path不是一个），同样利用上移下移调整这句“%USERPROFILE%\AppData\Local\Microsoft\WindowsApps” 的优先级，然后点击确定。


（4）在cmd中测试，终于可以正常运行了！

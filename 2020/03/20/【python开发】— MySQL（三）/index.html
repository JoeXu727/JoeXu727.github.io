【python开发】— MySQL（三）

一、视图

视图是一个虚拟表（非真实存在），其本质是【根据SQL语句获取动态的数据集，并为其命名】，用户使用时只需使用【名称】即可获取结果集，并可以将其当作表来使用。

1、创建视图
格式：CREATE VIEW 视图名称 AS SQL语句

2、删除视图
格式：DROP VIEW 视图名称

3、修改视图
格式：ALTER VIEW 视图名称 AS SQL语句

4、使用视图
使用视图时，将其当作表进行操作即可，由于视图是虚拟表，所以无法使用其对真实表进行创建、更新和删除操作，仅能做查询用。

二、触发器
